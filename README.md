# 🤖 AI Student Employee

An intelligent AI-powered personal assistant for students that monitors emails and files, generates daily plans, and takes actions with human approval.

## ✨ Features (Silver Tier)
- 📧 Gmail Watcher — monitors inbox automatically
- 📁 Filesystem Watcher — monitors Inbox/ folder
- 🧠 Plan Generator — creates daily Plan.md using Gemini AI
- ✅ HITL Approval — asks permission before sensitive actions
- 📤 MCP Email Sender — sends emails via Gmail API
- ⏰ Scheduler — runs plan generator daily at 9am

## 🛠️ Tech Stack
Python • Gemini AI • Gmail API • MCP • Obsidian Vault

## 📁 Structure
- scripts/ — all agent scripts
- skills/ — SKILL.md docs for each script
- Inbox/ → Needs_Action/ → Done/
